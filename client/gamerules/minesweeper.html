Minesweeper is a simple yet popular game among office workers as well as kid. It was popularized by Microsoft by bringing os pre-installed with this game.

Game setup : 

1. There are some cells of matrix containing mines.

2. Other are numbered.

3. Numbered cells indicate the number of neighbor cells that have mine in it.

Rules :

1. You can click any cell you want.

2. If you click a cell with mine, you lose.

3. If you click a cell with a number other than 0, you uncover the cell.

4. If you click a cell with a number 0, any other non-mine cells in the vicinity with be uncovered recursively.

5. If you are able to uncover all the non-mine cells, you win.

6. You can always re-start the game by clicking the refresh button.
